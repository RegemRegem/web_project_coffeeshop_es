# Triple Espresso - Cafetería de Especialidad

## Descripción del Proyecto
**Triple Espresso** es una página web estática y responsiva diseñada para una cafetería de especialidad moderna. El sitio web funciona como una plataforma landing page optimizada que permite a los usuarios explorar la identidad de la marca, consultar la ubicación, los horarios de servicio y gestionar de forma intuitiva un formulario de reservación de mesas.


### Tecnologías Utilizadas
* **HTML5 Semántico:** Estructuración limpia del documento utilizando etiquetas nativas (`<header>`, `<main>`, `<section>`, `<footer>`, `<fieldset>`)
* **CSS3 Moderno:** Implementación de diseños avanzados mediante **Flexbox** para la alineación de componentes y un sistema de posicionamiento absoluto/relativo para elementos decorativos complejos.
* **Arquitectura de CSS Organizada:** Uso de nombres de clases significativos y modulares para facilitar el mantenimiento del código.
* **Interactividad Dinámica:** Creación de transiciones suaves (`transition`) para estados de interacción (`:hover` y `:focus`) en botones, enlaces de navegación y redes sociales.
* **Checkbox Personalizado:** Reemplazo visual de componentes nativos del navegador mediante técnicas avanzadas de selectores CSS (`:checked` y combinación de hermanos) para cumplir con requerimientos estrictos de diseño sin romper la accesibilidad.
* **Git & GitHub Pages:** Control de versiones del código fuente y despliegue continuo en servidores de producción estáticos.

## Planes de Mejora del Proyecto (Roadmap Futuro)
Transformar esta landing page en una aplicación web interactiva de punta a punta:

1.  **Backend Automatizado para Reservaciones:** Migrar el entorno estático para procesar el formulario de forma dinámica permitiendo validar la disponibilidad de mesas en tiempo real y conectarlo a un gestor de base de datos relacional como **SQL Server**.
2.  **Seguridad y Autenticación de Usuarios:** Implementar un panel de administración para que los empleados de la cafetería gestionen las mesas reservadas. Este módulo incluirá sesiones seguras con tiempos de expiración automáticos y un sistema de **Autenticación de Dos Factores (2FA/MFA)** para robustecer la seguridad del acceso administrativo.
3.  **Sistema de Notificaciones por Correo:** Integrar un script en el servidor que dispare correos automáticos de confirmación y recordatorio a los clientes.
